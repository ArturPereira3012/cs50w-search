# CS50W Search - Front-End Search Engine Interface

A front-end web interface designed to replicate the look and behavior of Google Search, Google Image Search, and Google's Advanced Search pages, built as part of Harvard's **CS50's Web Programming with Python and JavaScript** course.

## 🚀 Features

- **Google Search:** A main search page with a centralized search bar and rounded buttons, functioning exactly like Google's core query system.
- **Google Image Search:** A dedicated page that allows users to search specifically for images, redirecting the query directly to Google Images.
- **Advanced Search:** A comprehensive page matching Google's advanced options, allowing users to filter results by:
  - All of these words.
  - This exact word or phrase.
  - Any of these words.
  - None of these words.
- **"I'm Feeling Lucky" Button:** Consistent with Google's native feature, this button bypasses the normal search results page and takes the user directly to the first layout result (includes handling Google's redirect notices).
- **Responsive Design:** A clean, CSS-focused layout that mirrors Google's aesthetics, complete with hovering effects, precise alignments, and full responsiveness.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Advanced layout styling, custom components)
- **Integration:** Native Google Search Query parameters
